Умный калькулятор:
    Данные в калькулятор вводятся через нажатия кнопок, вычисление проводиться при нажатие "=".
    Аргументы математических операций записываются в скобках.
    Геометрические выражение возвращают  значения в радианах.
    График функции изображается с помощью точек.
Кредитный калькулятор: 
    Заполняется информация по кредиту затем, для проведения вычисления нажать "РАССЧИТАТЬ".
Депозитный калькулятор: 
    Заполняется информация по депозиту затем, для проведения вычисления нажать "РАССЧИТАТЬ".

Переключения между калькуляторами проводится через "Тип" в меню окна/экрана
